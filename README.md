# 💻Deadlock_Detector
## 🚀 소개
### 프로젝트 소개
데드락을 감지하는, 방법론 중 하나인 Graph를 이용하여 탐지하는 방법을 구현하였다.

### 주요 기능
1. 데드락을 의도적으로 발생시킴

2. 해당 데드락이 발생하면, 그래프에서 순환이 발생함.

3. DFS를 이용하여 그래프의 순환을 확인하고, 순환 발생 시 5초 대기 후 다시 그래프를 확인, 데드락이 발생한 부분을 출력함.

### 기술적 특징
이 프로젝트는 단 2개의 스레드만을 순회하여 탐지한다.

일정부분을 수정하면, 임의의 스레드 수에 맞는 탐지가 가능할 것으로 보인다.

## 💻 설치 방법
